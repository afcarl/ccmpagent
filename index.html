ccmpagent

ART framework 2.1.1 agent implenentation using B-trust and decision trees.

The CCMPAgent is a software agent based on the ART testbed. It consists of three components: agent framework, WEKA based decision tree library, and a bayesian trust model library. The software was developed by Catalin Patulea, Chris Fournier, Mike Lepard, Pierre Dinnissen for the Software Agents Design class at Carleton University in the fall of 2009.

Mor information can be found on the project:

  • Wiki;
  • Downloads;
  • Source; and
  • a blog posting on the bayestrustlib component.
